Drop your photos in this folder with these exact filenames and they'll
appear on the site automatically (no code changes needed):

  floor.jpg      → big hero image (the 7th floor / wide office shot)
  apartment.jpg  → the "Live" section (a private apartment)
  studio.jpg     → the "Work" section (the shared studio / desks)

Tips:
- Landscape orientation, at least ~1600px wide, looks best.
- Until a file exists here, the site shows a clean gradient placeholder.
- Keep filenames lowercase. JPG or PNG both work (update the extension
  in styles.css if you use .png).
